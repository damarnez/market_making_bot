# market_making_b

Simple trading bot, using the DeversiFi API. 
**ONLY FOR TEST PURPOSES**

### Example .env

```
INIT_ETH="10.000000000000000000"
INIT_USD="20000.00000"
API_PRECISION = "P0"
API_SYMBOL = "tETHUSD"
NUM_ORDERS = 5
RANGE_ORDERS = 0.05
```

### Start


```
npm i
npm start
```

### Launch test

```
npm test
```
